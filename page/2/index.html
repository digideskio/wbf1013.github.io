工作与生活如何平衡？

工作与生活如何平衡？

最近忙的有些不像话了，完全没时间可以让自己慢下来思考一些事情。一方面，最近一直感觉自己身体不舒服，一方面，工作上的压力越来越大，要承担的东西也越来越多。

生活

发现自己身体体质变差也有一段时间了，从一开始的小腿长时间有酸痛的感觉开始，一直都没有好转的意思。之间还出现了扁桃体发炎的事情，导致自己身体39度高烧的情况下，混了两天。但是小腿酸痛的情况一直都没有缓解，直到自己喝大量的水，现在每天的喝水量相比以前多了好几倍。效果还是有些，缓解了一些小腿的酸痛。要继续保持，多让自己的身体活动，多走路，无论怎么说，当初在美国亚马逊海淘Fitbit也是这个原因，让自己更清楚自己的身体状况，睡眠状况，运动状况等等。这样才能让自己尽量地更健康地生活。

生活？ 如何更加健康地生活？？？

工作

从项目重新分组之后就一直没有闲下来的时间，每天都要处理一些很杂的事情，刚刚分组的时候，只是负责处理网银相关的业务，接着又接手bug整理、分类跟踪的工作，到现在又开始负责账单线的问题。负责的东西变得越来越多，工作上的压力也越来越大，现在又要面临软件架构的重新改造，更加合理化地处理金融业务。在金融业务上容不得一点差错，有一个小小的差错都有可能是致命的。甚至会让公司因为一个小小的错误失去运营金融业务的牌照。所以在现在改造的过程中更多地去了解相关银行金融业务的知识，更快地吸收这些知识。

从最近发现问题，讨论问题的过程中，发现公司里有些人本来就是在一些像招行，广发等银行中工作，他们相对我们而已，对银行金融业务更加地了解。把他们作为获取业务知识的来源途径是非常好的。他们可以在我们设计改造的过程中提出一些更合理的意见。现在的系统因为以前有些流程或业务没搞清楚银行的业务是怎么走的，而把自己所想的作为一个正确的业务流程，导致现在很多问题的修复极其困难。面对像现在这样的情况，有些很无奈，而且现在历史数据中很大一部分数据的问题都是由历史原因而造成的。

这几天满脑子里都是想着这些业务和程序中的逻辑，到底是在那一步出现了这样的问题。晚上睡前想，早上醒来还是在想这些东西。完全没办法出现从这样的过程中脱身出来。

工作？ 如何更加有效地工作？？？

生活 + 工作 = ？

现在这样的日子过的，完全没有空余时间干些其他事情了，把自己死死地锁在其中。没有空余的时间做自己想去做的事情。学自己想学的东西，看自己喜欢的书，要怎么办呢？？？如何脱身？ 如何平衡？

Devils never rest

一个人

练习一个人 书名 看到就被吸引了，然后亚马逊下手 作者很文艺，我很喜欢作者内心的那份宁静。

我一个人吃饭 旅行

到处走走停停

也一个人看书 写信

自己对话谈心

依然是心内一片寂静，总是不由自主地坚定认为，这么些年，自己只有一个人，自己始终在一个人生活。仿佛那些热闹那些经久的陪伴，都同自己没有关系。

陶立夏的文字清淡而且干净，一眼望过去就是文字功底极为丰厚，知识储备很多，而且，对文字有着天然的敏感的类型。还有她一个人生活的笃定和从容，甚是欢喜。

这个骨子里都透着文艺和浪漫的姑娘，离开了工作和人群，过了一年闲居的生活，背着包到处走走停停，拍下一个戳入眼睛的风景，看书，写字，思考。

她写道：

“确实没有人随心所欲地活着，生命即是束缚，但我们却可以经由努力与取舍，让生活尽可能接近你想象中的模样。”

“不是怕来不及，是此时此刻，非如此不可。”

一个人，是孤独寂寞与辛苦难捱，而陶立夏眼中的一个人，不过是一种再正常不过的生活状态。

是不断去遇见不一样的风景，见到不一样面孔的人，是不断地对自己的内心世界进行推翻与重组，直到自己心中那个影子，长成坚强倔强的模样。

是在一次次告别中学会走得更加没有留恋，是在一次次期待与失落中学会了更好地爱自己。

很多时候，我总是会陷入突如其来的深深孤独感中，莫名地就会感觉自己被抛弃到世界边缘，觉得没有人爱自己，觉得这世界这么多人，可是你看，我哭的时候没有人替我擦干眼泪，我疼痛的时候没有人拥抱我，我觉得生无可恋的时候也没有人拉着我走出那个巨大的黑洞。

所以我同陶立夏一般，“总是在寻找陌生的，更广阔的水域，期望在陌生感中获得短暂平静。”把自己放逐到一个陌生的地方，投身到一个陌生的环境。

然后，终于，开始平静。

“练习一个人，发现等待和依靠徒有虚名。自此不会陷于厌倦，也不再责怪生活的遗憾。”

我们没有办法避免掉一些不开心的事，没有办法挽留一些逃不掉的失去，没有办法时时刻刻等待和依赖。

而成长的最终，都需要明白，任何人都是独立的个体，在这个宇宙中独自生活，边走边爱，有时候有人同行一段，但是，没有人，和你的终点永远一致。

练习一个人，需要强大的内心，和不断充盈自己生活的欲望。

于陶立夏，练习一个人是行走，阅读，写字，和爱，是不死的对自由的欲望。

于我，练习一个人是不再逃避，不再慌张，不再恐惧，是好好地爱自己，整理自己所有的过往，静静地前行。

如同陶立夏在书中所言：“生命里的驿路梨花，含风带露开过，也曾妖冶压弯枝头。但都会过去。”

只有努力过才知道：喜欢一个人，仅凭努力，怎么足够。

如果我喜欢你的安静。这是要到一定的年纪，才能领会的好。没有形容词，副词，没有注解。

我没有想要万水千山走遍，我只是想去看你一眼，然后道别。

无法给你我的心，那我余生的时间，你要不要？

不问前程， 但闻旧情

“等待”这门技艺的诀窍是：学会放弃。

这世间虽有千般好，但唯你最珍贵。

爱使人卑微，更让人坚强。我想拿这块冰，换你更冷的心。

保持清醒并不太难，只需要一点勇气与许多耐心。

当你决定将心放在他人手里，又如何讨价还价。

没有人随心所欲地活着，抱怨只会显得你演技太差。

我没有迷路，只是暂时决定不了要去哪里而已。

我们好好告别，各自珍重，为了来日再相见。

有些话反着说更容易，但不是所有人都有勇气听懂。

我多么喜欢这生命中无所求的，专心致志的片刻。

人生这么长的旅程，一走几十年，怕的不是累，是厌倦。

Devils never rest。

提额 APP

提额 APP

开始参与这个APP也是巧合，一个月之前，我还在忙于电信运营商的工作，上级就过来问我在之前公司有没有用过 html css js这些。在维库的时候，可是从前台到后台都是要全包的呀，因为项目人数真的是少的可怜啊，说多了都是泪啊，还是不说为好。我很果断地回复了之前用过。然后我就默默地被调走去做这个app的前端。我一听是前端，我当时有点懵，不知道该如何下手。很久没有这样开始做一个项目，而且是负责前端，而不是自己更熟悉的后端，接口之类的。

html css js虽然之前用过，但也只是简单的使用。由于之前的工作中，美工是会把所有的html和css写好再给我们去实现。没想到这次，我们的设计师只负责给我一个颜色宽度等一些参数，让我自己去搞整个页面的结构。一开始的时候，我感觉很郁闷，不知道该从什么地方开始下手。

可是人员就我一个去负责前端的工作，没办法，我只能硬着头皮上了，自己慢慢地去参考他们之前做的app，怎么去拆解图片上的原因，如何用div+css去布局页面的结构。

时间一点一点地过去， 我也慢慢地开始上手了，一些css 不懂的地方去查firefox的developer center 和 google查询如何解决问题。 当然最直接的就是找公司的那些纯前端的同事去解决问题。

phonegap作为APP的壳，嵌入到ios、android等，只能手机的原生项目中去编译生成软件项目。这样就可以简化app开发的工作量。一次开发多个app都可以运行。通过网页的自适应来解决多平台之间的问题，phonegap是ios下运行的效率比我自己想象中的要好，完全不用担心一些类似android会出现的问题。android在一些高配的手机上还是蛮流畅的，但是一到低配的手机，简直卡的一B啊。说多了都是泪，都不好意思说。在wp8上现在还没有用phonegap自己去实现，应该会比想象中的好一些。从之前的wp8项目看来，卡顿的问题故意是没办法解决的。phonegap估计也会回复你说他们已经尽力了。我们只能尽量去优化自己APP的性能，从而达到客户比较满意的状态。

提额APP是一个单页的APP，所有的页面都会通过js来加载到首页中。也就没有多余的url地址。整个项目就只有一个。项目中的地址无论你是怎么去切换。都不会改变页面的地址。只是页面一层一层地网上叠加。但是这样做就遇到了一些问题。就是css3中的transition的效果，不能对position：fixed的元素进行正常的切换效果。这两个之前存在一种无解的冲突，最后只能通过把元素换成position:absolute进行transition切换，然后再把那个元素clone到position:fixed的外层中。 这样研究营造出了一种固定定位的效果。这样就满足了原来所需的一些效果。

总结：

  1. APP项目 PhoneGap + html + css + js 服务端用 ASP .net ashx作为服务接口
  2. 增加了html css 盒模型的理解， 更加清楚 position在 网页上的一些使用
  3. 对APP的开发流程有了进一步的了解，熟悉。做web app的想法还可以继续坚持
  4. 作为前端的角色，去完成一个从无到有的项目，这种感觉还是不错的。

后会无期

后会无期

韩寒的作品大部分还是比较喜欢的，所以在这部电影上映之后立马去电影院观看。

这算是一部公路片吧。一直是在路上，经历着一系列的事情。故事的描述似乎平淡的出奇，毫无爆点。对于有经历的一些人来说，这部电影应该算是不错的，会很容易就击穿他们内心柔软的部分。对于一些毫无经历的人来说，这电影真TM的无聊，电影完全没有故事感，这样的电影就不该花钱看，烂片，负分。

然后这电影却是描述了80后一代，不甘于现状，而出去闯荡，经历，生活，释放自己的青春，最后回归于平凡的一部戏。韩寒用黑色幽默的方式来演绎这些。我觉得这些就足够花这点票价去看这电影。这样的电影还需要什么其他的东西去解释呢？这些足够了，没必要还要其他的东西。

生活在一直延续，你跟你的很多朋友都会后会无期，也许你今天见面了，明天你就见不到了。这些事情在很多人的身上都在延续。你经历过了，你就会被他的这种平淡的叙述打动。

在电影院里，笑声是一阵一阵的，都被那些所谓的经典台词所击中。回想起来，我笑得不多。不是我笑点低，只是我在尽量地去感受电影中的人物为什么会有这样的表现，为什么会说这样的话。在袁泉（刘莺莺）和浩汉讲述为何他们会进行结对的时候，我那个时候我感觉特别的心酸。一个父亲因为种种原因不能跟自己的儿子见面，只能依靠别人来完成自己的愿望。阿拉斯加（犬）的乱入简直萌化了所有女生的心吧。

旅行者这样的称号在这个电影中显示的比较好，钟汉良是一个完整的旅行者，带着自己的梦想一直都是在闯南走北。“我所做的事情，全世界中有一个明白的人就好了。”（懒得去翻台词，差不多就这意思） 这句台词击中了我。我一直都觉得很多人都不了解我，不理解我。相信那个人会来的，我在这里等你。

赞一下 电影主题曲 《平凡之路》， 朴树，从小就喜欢的歌手。歌词便深深打动了我的内心。我经历过从乐观到忧郁，到抑郁，再从抑郁尽量走出来， 朴树也经历过这些。 我们感同身受。这些都需要经历来解释。

后会无期 对于韩寒的处女作来说，是很不错的，用黑色幽默的方式演绎了那些80后一代的经历。只有经历才可以让人走的更好，没有经历的人生是多么无趣。

送上 《平凡之路》

  1. 感觉到自己身体有不适，晚上睡眠质量不好，没有办法有数据来说明。

  5. 用数据促使自己生活的更加健康。改善自己的身体不适的状况。

我想找个可以说话的人一起生活

我想找个可以说话的人一起生活

每次放假回家都会遇到那件事情——“什么时候找女朋友”。每次都是随便应付一下，然后对此就不发表任何言论。

要说到现在这个年龄了，完全不去考虑这方面的事情是不可能的。只是现在怎么也找不到有这样一个人，可以随时随地的说说话，而不是两个人面对面的坐在一起，却手里一直拿着手机。两人之间只有只字片语。显然我没办法接受这样的生活。

为此，我挑剔的生活着，直到有那么一个人出现，两人在一起的时候，可以放下身边的手机、电脑等等。不要让这些电子设备影响到生活。我不能否认有些智能设备可以改善我们的生活，带给我们健康。比如，最近在美国亚马逊上海淘的Fitbit Flex智能手环，我相信它可以改善我的生活，跟踪我生活的规律，从而让自己更加健康的生活。

现在我们的生活，充斥着各种社交网络，微博，微信，朋友圈等等，里面的好友很多，但是可以相互聊的人很少。微博和朋友圈充斥着各种生活的垃圾，很多信息都是毫无意义，而且对朋友之间的了解完全没有帮助。大家都在晒一些毫无营养的东西，博存在感。为此，我关闭了朋友圈。

看了英国作家的给低头族的诗，感触就更深了。我们每天都在低着头，玩着手机等设备，减少和其他人面对面的交流，从而产生了人与人之间的隔阂。面对着冷冰冰的电子设备，你无法感受到对面的人真实的感受。如果对方给你发了一个开心的表情，在冷冰冰的电子设备面前，你只能当作对方现在是处在开心的状态。而无法感受到也许对方是在苦笑。每天都需要面对冷冰冰的电子设备去猜测，这样很累。

这样的生活，我真的无法接受，现在也慢慢地想从这样的生活中脱离出来。不要每次都被冷冰冰的电子设备绑架，所以有的时候，我会不带手机或者带了手机出去，也许是处于关机状态。用自己的身体去感受周围的生活，而不是去假想。

好像有些扯远了，有些凑字数的嫌疑。回到正题上，我想找个可以说话的人一起生活。 这也许会成为我生活的目标。

来，我们聊聊吧……

不挑剔终将成为最挑剔的选择

心绪乱飞，无休无止。 ありがとう-大橋卓弥

自己原本是一个从来都不挑剔的人，享受着简单而不挑剔的生活。对任何事物都没有太多的要求，够用就好。在其他人看来没有一点上进的意思，过着一切都不在乎的生活。正因为自己的不在乎，自己的体重也上升了，飙泪。。。真是心宽体胖的例子啊。。。。

不知从何开始，自己已经被一种文化慢慢地改变。也许就是从大学的日语课开始，自己已经无形地拉近距离。从而开始接触日本的一些东西，首先接触的当然是日本最优秀的漫画，《火影忍者》，死神，柯南等等优秀的漫画。当然也会看一些日剧。 日剧每一部都非常精致，每次都会把某个细小的东西讲的很透彻朴实，而不像大陆的电视剧，各种神化不堪入目，没有一点营养。接着慢慢接触到了日本的一些作家，歌手，设计。

从电视剧《一公升的眼睛》认识了泽尻英龙华，那时她还叫泽尻绘理香，认识了锦户亮。从零秒出手认识了山下智久。从神探伽里略认识福山雅治 和 东野圭吾 。 从 1Q84认识到 村上春树。慢慢地一点一点地去了解日本的文化。然后去接触了一些设计的东西，这下真的完了，被他们这种简约的设计沦陷了。自己也开始去适应这种简约的东西，把繁华的东西踢出了自己的生活。

自从简约的设计理念进入我的生活之后，我丢弃了很多繁琐的东西，一切都想要从简。每个使用的东西都想要简化。从而让自己变的越来越挑剔。越来越难以接受自己不喜欢的东西。

生活美好，过挑剔的生活。

奇特的一生

奇特的一生

人的一生或精彩，或虚度。怎样过奇特的一生？在看完《奇特的一生》之后，莫名的会去思考这样的问题？去做一些令自己疯狂的事，还是去做一些令自己幸福的事？人总是在期待一些美好的事物会到来～有的时候，幸福来的太突然……

好，回到《奇特的一生》的这本书，主人公是一位俄国的逝者——柳比歇夫。他的一生就是奇特的一生，在他去世之后，才开始有人对他有一些尊敬的看法。他是一位科学家，那么他是怎样过的这奇特的一生呢？

首先，我们应该思考这样一个问题，怎样生活得更好？柳比歇夫就有自己的一套方法，而且他坚持了一生，从而让他具有这奇特的一生，让他人觉得不可思议。他的方法就是“时间统计法”，他可以不用借助其他计时器，可以准确地判断出所花的时间。经过不断地记录，去总结，不断地调整自己，最后可以说他已经和时间同在。他用时间开销日记记录了他的一生。每个月都会画一些图表，做总结，到年终又会根据月总结，做一份年度总结。

他的成果令人惊叹，生前发表了70来部学术著作。其中有分散分析、生物分类学、昆虫学方面的经典著作。还有其他一些回忆录，和大量的信件。我无法想象他究竟是怎样过完这一生的，他这一生只在做他想做的事情，错过了很多可以像其他一些科学家获得社会好评的机会，或者说让自己和家人过上更好的生活的机会，他的生活极其简朴，超过最低水平的东西，他都不接受。不被外界所诱惑，一直过着自己想要的生活，带着自己远大的目标一直走下去。

摘录：

  1. 我要谈的是一位当代的杰出人物，是他们没有听说过的。我要描绘一位精神上的英雄，他的道德信条是如此崇高，以致如今看来似乎是不合时宜了。他的一生，表面上再平凡不过，从某几点上说，竟是“不如意事十有八九”。以小市民的眼光看，他是个典型的不走运的人。然而就生活的内涵而论，这个心灵和谐，很幸福，而且他的幸福是一种最高级的幸福。
  2. 柳比歇夫有他的崇拜者、拥护者、景仰者，也有读者。他没有学生，只有私淑弟子。也就是说，他并没有教他们，是他们向他学习。学习什么呢？很难说。主要是学习应当怎样生活、怎样思考吧。似乎咱们总算找到了一个人，他知道他活着是为了什么,有什么目的……仿佛他有什么崇高的目的，甚至可能领悟了他存在的意义。不仅仅是过一种合乎道德的生活，不仅仅是问心无愧地工作，似乎他明了他所作所为的内涵。
  3. 他的工作方法是一个创举，不问他其余的工作和研究如何，这种工作方法是独立存在的。从表面上看，这是纯技术性的工作方法，一点也不起眼。它是自然而然形成的，但几十年来它取得了精神上的力量。它成为柳比歇夫生活的骨架，不仅保证了最高的效率，并且保证了最旺盛的生命力。
  4. 我们一生的时间，大部分用于错误及种种恶行；很大一部分虚抛浪掷，无所事事。我们整个一生，几乎都没有用来干应当干的事。
  5. 大家知道，幸福的人是不看表的；反过来说也对，不看表的人是幸福的。但是，柳比歇夫不是由于工作上的关系，不是出于不得已，而是自愿地担当起“看表”的苦差使。
  6. 柳比歇夫如此忠于自己的方法，这本身便是一个独特的现象；这样的日记，单单它的存在，说不定便是独一无二的。
  7. 最好不是去震惊世界，而是像易卜生所说的，生活在世界上。
  8. 他对“时间下脚料”的利用，考虑的无微不至。出门旅行，他看小部头的书，学习外语。举个例子，英语，他就是主要利用“时间下脚料”学会的。
  9. 计划就是挑选时间、规定节律，使一切都各得其所。
  10. 这个方法之所以能够存在，是依靠经常的计算和检查。没有计算的计划是盲目的计划，就像某些研究所那样，光会做计划，却不去操心这计划能不能完成。
  11. 应当学会计算一切时间。

惊叹怎么会有人过这样的一生。当然，我不会去过这样的一生，别人的奇特自己是没办法去复制的。需要考虑的是自己怎样去过奇特的一生？

你看，情绪泛滥了

想写些东西，却不知道怎么写。最近的生活过的很古怪，总是会有一些奇怪的想法出现，然后去做一些令人觉得古怪，无聊甚至蛋疼的事。就拿昨天无缘无故地情绪泛滥，想走走。却又不知该从哪里走到哪里。上班时间很长，下班时间也比较晚，自己要是出去走走，那回到家就更晚了吧。那要怎么办呢？总得有个选择，那就提前下公交车走回家好了。这样应该会是一个两全的选择。有些人不理解我，我也不想多去解释，默默的做着自己愿意去做的事。不期望别人是否可以认同我。也罢，不可能让所有人都理解我。做些古怪的事情，反正已经不是第一次了。

想想那些古怪奇葩的想法和行动，觉得这样so navie。

我知道我自己一直封闭着自己，不愿去接受其他人的一些什么东西。一直都是以一个旁观者，局外人的身份在生活。这样的生活如此美好，而且没有什么压力可言。那不去参与的生活到底还可以过多久呢？ 活在自己的世界里总是很美好的。走走吧，说不定走到谁的生活里了。

你好，生活！想想当初的微笑。

来，我们来谈谈生活。

早上公交车上 ，你的情绪又泛滥了吧， 看个公益广告，你居然开始两眼含泪了。 在你发现你快哭的那瞬间，你意识到是在公共场合，要是哭出来，会让周围的人觉得很奇怪。 你匆忙地看看周围，以忍住自己的的泪水。 你还无法把握你的情绪， 你为了封闭自己，你努力地把你感性的一面藏起来。 可是在潜意识下，你的感性很快就张扬起来。 你为了让别人看到你的理性。 你慢慢地变得冷漠， 变得不在乎， 不在乎那些爱你的人对你的爱。 你放弃了那些。

唉， 算了 ， 情绪又要泛滥了。 你看， 那里有个坑。。。 哪里有啊？
