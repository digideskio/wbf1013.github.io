Fitbit Flex

Fitbit Flex 使用

7月4日，收到了在美国亚马逊上海淘的Fitbit Flex。首先谈谈这价格，在美国亚马逊上购买时的价格是98美元，下订单过后没几天，京东开始首发出售，价格定在898元。相对国内的价格，美国亚马逊海淘回来还是便宜不少的。相对国内的一些智能手环，在美国亚马逊上海淘fitbit flex还是非常有优势，毕竟fitbit flex现在是大品牌的穿戴设备供应商。

购买fitbit flex的缘由：

  1. 感觉到自己身体有不适，晚上睡眠质量不好，没有办法有数据来说明。
  2. 在知乎上找了一些可以监测身体一些运动数据的回答，fitbit flex和UP手环的使用者比较多，这两个者的功能最强大。
  3. 虚荣心～

fitbit flex使用心得：

  1. 对模式的设置上遇到了一些问题，轻轻敲击并没有像官网帮助文档所说的那样，也可能是我使用的不当。
  2. 监测还是很灵敏的，每项数据都显示的很清晰。
  3. 每天记录自己的走路步数，运动时间，休息睡眠时间，从而使得自己去慢慢地会调整自己的状态。
  4. 工作日目前基本上可以完成1w步的运动量，非工作日完成不了，有待加强。
  5. 用数据促使自己生活的更加健康。改善自己的身体不适的状况。

用运动改善自己，让自己从阴暗中走向阳光～
